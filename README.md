# testingSDCtools
This is an example of application of the R-packages included in SDC-tools (https://github.com/sdcTools) to census data of high resolution (small geospatial areas).
A Working Paper has been published based on the results for Icelandic data, at: http://hagstofan.s3.amazonaws.com/media/public/2020/e9ea7160-5032-4580-9297-7b3b3cb634da.pdf
